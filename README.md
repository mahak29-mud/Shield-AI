# Shield-AI
Shield AI is an AI powered cyberbullying detection web application built using Flask, Machine Learning, Scikit learn, and SQLite. It analyzes text messages and predicts whether the content is safe or bullying using an SVM model with TF IDF vectorization, along with secure authentication and scan history tracking.

# Project Objective
The objective of this project is to detect cyberbullying text automatically and promote safer online communication using Machine Learning and web technologies.

# Features
- User Registration and Login
- Secure Password Hashing
- AI Based Cyberbullying Detection
- Confidence Score Prediction
- User Scan History
- SQLite Database Integration
- Responsive Web Interface
